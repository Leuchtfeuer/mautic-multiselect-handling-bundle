### Usage

1. Install plugin.
2. Folder Name is MauticContactSegmentsBundle
3. Set up custom field(s) with multiselect type, which will contain a list of Segments for synchronization. Needs to have the same alias
4. Add form action "Modify Segment Membership based on Multiselect"

After form is submitted the Contact will be added/removed to/from segments according to the multiselect selected in the form action.
